# Mallow へようこそ

軽量で **WYSIWYG**(見たまま編集)の macOS 向け Markdown エディタです。

## まずは試してみましょう

- Markdown を入力するとその場で書式が適用されます — **太字**、*斜体*、`コード`。
- **⌘F** で検索・置換ができます。
- 右上の **T** ボタンでテキストスタイル、**ⓘ** で統計と目次を表示できます。
- **⌘E** で PDF に書き出せます。

> 一度保存すると、その後の編集は自動的に保存されます — ⌘S を押し続ける必要はありません。
