# Welcome to Mallow

A lightweight, **WYSIWYG** Markdown editor for macOS.

## A few things to try

- Type Markdown and watch it format as you go — **bold**, *italic*, `code`.
- Press **⌘F** to find and replace.
- Use the **T** button (top-right) for text styles, or **ⓘ** for stats and an outline.
- Export to PDF with **⌘E**.

> Once you save a file, your edits autosave automatically — no need to keep pressing ⌘S.

Select all and delete this text to start writing. ✍️
